## Setup Instruction:
### Backend:
  1. Go to Backend: `cd server`
  2. Insert all the  credentials:`cp .env.example .env`  : # Add a Postgres local setup password here.
  3. Install all the required libraried:`npm i` 
  4. Run the Server:`npm start`


### Frontend:
  1. Go to Frontend: `cd server`
  2. Insert all the  credentials:`cp .env.example .env`  
  3. Install all the required libraried:`npm i` 
  4. Run the Server:`npm run dev`


## Tech Stack:
1. Backend: Postgres, Node.js, Express.js
2. Frontend: React.js, Tailwind css, Context.api
